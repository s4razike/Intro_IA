¿Cómo funciona?


Al momento de reconocer posiciones Teachable Machine se basa en primer lugar en el reconocimiento de una imagen.

a partir de esto se crea una función canvas la que reconoce las diferentes extremidades a partir de un

modelo basado en el esqueleto humano.

Gracias a esta guia es que el modelo la toma para hacer una estimación de la posición hecha

por medio de grandes diferencias y similitudes que le permiten diferenciar una posición de otra

Teniendo esto como base y la música como el centro decidimos hacer un modelo de reconocimiento en base a la posición

al tocar un instrumento en especifico. Esto va desde grandes rasgos como el tocar parado o sentado

hasta la posición de las manos como datos más especificos

Position Model

Se capaz de reconocer un instrumento según la posición en la que lo tocas
